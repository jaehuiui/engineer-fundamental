# Project Baseline Setup Plan

## 목표

Staff Engineer 학습 노트 레포지토리의 프로젝트 베이스라인을 구성한다.

## 생성할 파일 목록

### 1. Git 초기화

- `git init` 실행
- `.gitignore` 생성 (macOS, IDE 관련)

### 2. CLAUDE.md

프로젝트 컨텍스트 파일. 다음 내용 포함:

- 프로젝트 목적: Staff FE Engineer 학습 노트
- 언어 규칙: 제목/키워드는 영어, 설명은 한국어
- 폴더 구조 규칙: 질문 시 필요한 폴더를 그때그때 생성
- 문서 작성 규칙: md 형식, 각 주제별 폴더 하위에 파일 생성
- 답변 규칙: 질문에 답변 후 해당 내용을 학습 노트로 정리

### 3. README.md

- 레포 소개
- 폴더 구조 (현재는 비어있음, 학습하면서 채워짐)
- 사용법 안내

### 4. Memory 설정

- user memory: Staff FE Engineer 준비 중인 프론트엔드 개발자
- feedback memory: 혼합 언어 스타일 (영어 제목 + 한국어 설명)
- project memory: 학습 노트 레포 목적과 운영 방식

## 생성하지 않는 것

- 빈 주제 폴더들 (Minimal 방식이므로 질문 시 생성)
- 템플릿 파일들 (불필요한 boilerplate 지양)
